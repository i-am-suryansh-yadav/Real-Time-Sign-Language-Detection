Sign Language Detection

Features

  • Robust Design: Built with efficient computer vision pipelines for reliable performance.
  • Prediction Confidence: Displays real-time confidence percentage for each detected letter.
  • Hindi Mapping: Maps English letters to corresponding Hindi characters for ISL.
  • FPS Detector: Monitors and displays frames per second for smooth operation.
  • Word Builder: Constructs words from sequential sign predictions.
  • Two-Hand Support: Detects gestures using one or two hands for flexibility.
  • Demo Recording: Easily record sessions for demonstrations or analysis.
  • Browser Accessibility: Run detection directly in your web browser without installations.

About

SignSync is a real-time Indian Sign Language (ISL) detection system designed to bridge the gap between sign language users and spoken communication. It uses advanced AI and computer vision to recognize A-Z gestures with high accuracy.

Developed by Suryansh Yadav, a passionate developer focused on accessible technology.

Contact: suryansh.1251010449@vit.edu

This project leverages open-source libraries like MediaPipe, OpenCV, and scikit-learn to provide a seamless experience. Future updates may include voice output and expanded gesture support.
